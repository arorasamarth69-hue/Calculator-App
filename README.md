This is the first milestone of building an advance calculation app, the first version has basic addition, subtraction, multiplication and division functionalities.
In the next projects, the goal is to make the calculator app somewhat limitless, and add more logical functions like exponentiation and calculus.
This project uses as of now, math.js.
Created the design with the HTML Table where first is holding the input field with id="result" and the rest is filled with input button.
With every click of the button, it displays the respective value of the button to the input field by using the function dis().
myFunction() is used to set the value pressed from the keyboard to the same input field.
Calculation of the numbers can be done by both the "Enter" key as well as the "=" button on the Calculator UI.
The solve() function is evaluating the result with the math.evaluate() and displays the final answer to the input field with id="result".
clr() function is also defined to clear the input field.
